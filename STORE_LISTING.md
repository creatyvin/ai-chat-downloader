## Name
AI Chat Downloader

## Short Description (132 characters max)
Export your AI conversations from ChatGPT, Claude, Gemini, Perplexity, Grok, and DeepSeek to clean Markdown files.

## Detailed Description

**Save your AI conversations forever.**

AI Chat Downloader lets you export chat conversations from popular AI platforms to well-formatted Markdown files. Perfect for archiving important discussions, creating documentation, or sharing AI-generated content.

### Supported Platforms
- ChatGPT (chatgpt.com)
- Claude (claude.ai)
- Gemini (gemini.google.com)
- Perplexity (perplexity.ai)
- Grok (x.com)
- DeepSeek (chat.deepseek.com)

### Features
- **One-Click Export** — Download any conversation with a single click
- **Clean Markdown** — Properly formatted output with headers, code blocks, and lists
- **Auto-Scroll** — Automatically loads full chat history before exporting
- **Code Preservation** — Maintains syntax highlighting language tags
- **Artifact Support** — Extracts Claude artifacts and canvas content
- **Keyboard Shortcut** — Press Ctrl+Shift+D (Cmd+Shift+D on Mac) for quick access
- **Custom Folder** — Configure your download location in settings
- **Privacy First** — Works entirely offline, no data sent anywhere

### How to Use
1. Navigate to any supported AI chat
2. Click the extension icon or press Ctrl+Shift+D
3. The extension auto-scrolls to load full history
4. Your chat downloads as a Markdown file

### Output Format
Each exported file includes:
- Chat title
- Export date
- All messages with clear User/AI role labels
- Preserved code blocks with language tags
- Links and references

### Privacy
This extension:
- Does NOT collect any personal data
- Does NOT send data to external servers
- Does NOT require an account
- Works 100% locally in your browser
